# Main Inventory App

## Current Status:

- **Backend:** Spring Boot with JWT Auth & MySQL. Endpoints at `/api/auth` and `/api/items`.
- **Frontend:** Vue 3 + Tailwind CSS initialized. Login page designed but needs final testing.
- **Database:** MySQL configured.

## How to resume:

1. Backend: `./mvnw spring-boot:run`
2. Frontend: `cd frontend` -> `npm install` -> `npm run dev`
